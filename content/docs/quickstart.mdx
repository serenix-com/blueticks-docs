follow each language's idiomatic naming conventions.

## 1. Install

<Tabs groupId="sdk" persist items={['Python', 'Node.js', 'PHP', 'Ruby', 'Go']}>
  <Tab value="Python">

```bash
pip install blueticks
```

  </Tab>
  <Tab value="Node.js">

```bash
npm install blueticks
```

  </Tab>
  <Tab value="PHP">

```bash
composer require blueticks/blueticks guzzlehttp/guzzle
```

PHP uses PSR-18 HTTP clients. Install any concrete client (Guzzle shown);
the SDK discovers it at runtime.

  </Tab>
  <Tab value="Ruby">

```bash
gem install blueticks
```

  </Tab>
  <Tab value="Go">

```bash
go get github.com/serenix-com/blueticks-go
```

  </Tab>
</Tabs>

## 2. Set your API key

Export your key as an environment variable so the SDK picks it up
automatically:

```bash
export BLUETICKS_API_KEY="bt_live_..."
```

See [Authentication](/docs/authentication) for how to obtain a key.

## 3. Ping the API

<Tabs groupId="sdk" persist items={['Python', 'Node.js', 'PHP', 'Ruby', 'Go']}>
  <Tab value="Python">

```python
from blueticks import Blueticks

client = Blueticks()
ping = client.ping()
print(ping.account_id, ping.whatsapp_connections)
```

  </Tab>
  <Tab value="Node.js">

```typescript
import { Blueticks } from "blueticks";

const client = new Blueticks();
const ping = await client.ping();
console.log(ping.account_id, ping.whatsapp_connections);
```

  </Tab>
  <Tab value="PHP">

```php
<?php
require 'vendor/autoload.php';

use Blueticks\Blueticks;

$client = new Blueticks();
$ping = $client->ping();
echo $ping->account_id, ' ', count($ping->whatsapp_connections), " connection(s)\n";
```

  </Tab>
  <Tab value="Ruby">

```ruby
require "blueticks"

client = Blueticks::Client.new # reads BLUETICKS_API_KEY
ping = client.ping.retrieve
puts ping.account_id
```

  </Tab>
  <Tab value="Go">

```go
package main

import (
	"context"
	"fmt"
	"log"

	blueticks "github.com/serenix-com/blueticks-go"
)

func main() {
	c, err := blueticks.NewClient() // reads BLUETICKS_API_KEY
	if err != nil {
		log.Fatal(err)
	}
	ping, err := c.Ping.Retrieve(context.Background())
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println(ping.AccountID)
}
```

  </Tab>
</Tabs>

A successful response looks like:

```json
{
  "api": "ok",
  "account_id": "acct_abc123",
  "whatsapp_connections": [
    { "id": "engine_01h7...", "type": "gateway", "connected": true }
  ]
}
```

`api: "ok"` confirms your key works. `whatsapp_connections` lists the
WhatsApp engines currently connected for your account — an empty array
means no WhatsApp is connected yet (not an error; the `message` field
explains it).
