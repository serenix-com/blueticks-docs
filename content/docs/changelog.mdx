The API and each official SDK track versions independently.

## API specification

### v1.0.0 — 2026-04-23

- Initial public release.
- Endpoints: `GET /v1/ping`, `GET /v1/account`.
- Error envelope contract locked (see [Errors & retries](/docs/errors)).

## SDKs

Each SDK maintains its own changelog on GitHub:

- [Python — `blueticks`](https://github.com/serenix-com/blueticks-python/blob/main/CHANGELOG.md)
- [Node.js — `blueticks`](https://github.com/serenix-com/blueticks-node/blob/main/CHANGELOG.md)
- [PHP — `blueticks/blueticks`](https://github.com/serenix-com/blueticks-php/blob/main/CHANGELOG.md)
