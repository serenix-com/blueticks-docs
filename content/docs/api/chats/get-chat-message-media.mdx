title: Get chat-message media
description: >-
  Download the media attached to a WhatsApp message (image, video, document,
  audio).

  Returns either a hosted URL (`url`) or inline `data_base64`, plus mimetype +
  filename.


  CAVEAT: for own-sent newsletter media (messages you sent to a `@newsletter`
  chat), the bytes returned are a WA-generated preview JPEG (~7KB) rather than
  the original you uploaded — WA releases the original blob from memory
  immediately after the unencrypted upload. The response will include
  `original_quality:false` when this fallback is in effect; the caller can warn
  the user or decide to retry from a different source.


  Requires `chats:read`.
