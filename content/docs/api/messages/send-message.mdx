The body is a discriminated union — set the `type` field to one of `text`, `media`, or `poll`.

**Variants:**

- `type: "text"` — required `text` (1–4096 chars). Example body:
  ```json
  {"type":"text","to":"+15551234567","text":"hello"}
  ```
- `type: "media"` — required `media.url` (HTTPS). Optional `media.kind` (`image` · `video` · `audio` · `document` · `sticker` · `voice`; auto-detected when omitted), `media.caption`, `media.filename`. Example:
  ```json
  {"type":"media","to":"+15551234567","media":{"url":"https://cdn.example.com/x.pdf","kind":"document","filename":"receipt.pdf"}}
  ```
- `type: "poll"` — required `poll.question` and `poll.options` (2–12 items). Optional `poll.allow_multiple` (default false). Example:
  ```json
  {"type":"poll","to":"+15551234567","poll":{"question":"Pizza?","options":["Yes","No"]}}
  ```

All variants accept optional `send_at` (ISO 8601 ≥10s future, ≤365d), `from` (E.164 sender for multi-session workspaces), and `reply_to` (wire `key` of a prior message to quote).
