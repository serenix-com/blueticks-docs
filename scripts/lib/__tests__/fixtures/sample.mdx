## Send a poll

<Tabs groupId="sdk" persist items={['Python', 'Node.js', 'PHP', 'cURL']}>

```python
msg = bt.messages.send(
    to="+972",
    type="poll",
    poll={"question": "Q?", "options": ["Yes", "No"], "allow_multiple": False},
)
```

  </Tab>
  <Tab value="Node.js">

```ts
const msg = await bt.messages.send({
  to: '+972',
  type: 'poll',
  poll: { question: 'Q?', options: ['Yes', 'No'], allow_multiple: false },
});
```
